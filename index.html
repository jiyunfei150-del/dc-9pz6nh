🧠记忆系统 · 机器要求
  • CPU 4核+，内存 8GB+，存储 50GB+
  • 操作系统：Ubuntu 20.04+ 推荐
  • 需要出网访问 LLM API（Anthropic / OpenAI）
  • 不需要访问内网业务数据库
  • Java 17（Neo4j 依赖），Python 3.10+
  • 服务端口：8100（仅对智能问数平台开放）
📡接口规范
  • 调用方：智能问数平台 → POST /query
  • 入参：question / project / user_id
  • 出参：sql / intent / metrics[]
  • 管理入口：GET /admin（指标录入 Web UI）
  • 同步接口：POST /sync（刷新 Weaviate + Neo4j）
  • 无数据库出口，防火墙可封闭内网端口
